kedge infrastructure provider

This binary is a kedge provider that brokers application templates from a central kro (Kube Resource Orchestrator) cluster. It is designed to mount inside the kedge portal as a custom element (<kedge-provider-infrastructure>), loaded from /main.js. You're seeing this fallback page because you opened the provider URL directly.

Visit the portal at /ui/providers to see it mounted natively, or fetch /main.js to inspect the custom-element bundle.
